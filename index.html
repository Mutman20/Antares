AntaOS Section

Welcome to my ongoing project a fully functional operating system crafted from scratch using C# and leveraging the powerful capabilities of the Cosmos OS library. Currently, it exists as an executable file, serving as a prototype for the ambitious vision of a robust operating system in the future.

This project represents a culmination of rigorous exploration and experimentation in the realm of computer science. Through meticulous coding and testing, I aim to create a platform that not only demonstrates technical proficiency but also fosters a deeper understanding of operating system architecture and design principles.

Within this operating system, you can expect to find a range of core functionalities, including a fully operational file system and a command-line interface. These elements are meticulously implemented to provide users with a seamless and intuitive computing experience.

In its current state, the operating system is text-based, but future iterations will incorporate graphical elements to enhance user interaction. Additionally, it includes various tools and utilities, such as a calculator, odd or even checker, and number guessing game, alongside basic games. These features aim to provide users with both practical and entertaining applications.

As this project evolves, I invite you to join me on this journey of discovery and development. Feel free to explore the current iteration of the operating system and provide feedback that will contribute to its ongoing refinement and improvement. You can download the current executable version using the buttons below.

AntaOS Wiki Section

Welcome to AntaOS! Let's get started.

1. Installing AntaOS
To install AntaOS, please locate the download links above this section. You'll find four options to choose from. I highly recommend selecting version 3.0, as it's the most advanced and stable version available. However, feel free to choose another version if you prefer.

2. Version 1.6
Choosing version 1.6 will lead you to a simple command prompt where you can enter your username and password. Additionally, you'll be prompted to decide whether you want to input your password every time you log in, with options 'y' for yes and 'n' for no. Once you've completed this step, you're all set to start using AntaOS. I suggest typing "help" to explore all available commands.

3. Version 2.2
Opting for version 2.2 will also bring you to a command prompt similar to version 1.6. However, please note that this version has been canceled, limiting its functionality. Nonetheless, you can still access basic commands using the "help" feature.

4. Version 3.0 (BETA)
If you've chosen Version 3.0, congratulations! This version, like version 2.2, is written in C#, offering enhanced features compared to version 1.6, which was written in Lua. It presents a graphical user interface (UI) for a more intuitive experience. Upon startup, you'll be prompted to create a new username and password (ensure it's secure). Once entered, you'll be greeted with a user-friendly menu where you can execute commands simply by typing the corresponding number or name. The commands will then run automatically.
